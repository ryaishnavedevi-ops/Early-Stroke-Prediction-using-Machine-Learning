# Early-Stroke-Prediction-using-Machine-Learning
Stroke Risk Prediction System using Machine Learning that analyzes patient health data such as age, BMI, glucose level, hypertension, heart disease, and smoking status. The model uses SMOTE to handle imbalance and predicts stroke risk through a user-friendly web interface.
